# Converter

Grundstruktur für den NWG-Converter.

## Projektstruktur

- `src/` – Anwendungscode
- `tests/` – Tests
- `requirements.txt` – Python-Abhängigkeiten

## Schnellstart

1. Python-Umgebung erstellen/aktivieren
2. Abhängigkeiten installieren:
   - `pip install -r requirements.txt`
3. App starten:
   - `python src/main.py`
